## Лабораторная работа 3. End-To-End Tests

### 1. Описание проекта

**Проект**: 

В качестве объекта тестирования был выбран мой пет-проект, содержащий пользовательский интерфейс и взаимодействующий с API

### 2. Анализ пользовательских сценариев

**Ключевые точки интеграции**:

- Авторизация пользователя

  Критическая точка входа в систему, ошибки на этапе входа делают приложение недоступным;

- Просмотр основной страницы после входа

  Сценарий проверяет корректность маршрутизации и отображения интерфейса;

- Обработка ошибок при неверном вводе

  Необходимо убедиться, что система корректно реагирует на неверные данные и отображает сообщение об ошибке.


### 3. Примеры написанных тестов

Для написания E2E‑тестов я использовала Cypress, так как он легко интегрируется с Vue

- Успешная авторизация
  
  ```javascript
    describe('Successful Login', () => {
    it('Авторизация с корректными данными', () => {
    cy.visit('http://localhost:8080/login');
    cy.get('input[name="login"]').type('test_user');
    cy.get('input[name="password"]').type('test_password');
    cy.contains('Войти').click();
    
    
    cy.url().should('include', '/main');
    cy.contains('Добро пожаловать').should('be.visible');
    });
    });
  ```

- Ошибка авторизации

  ```javascript
  describe('Login Error Handling', () => {
  it('Отображение ошибки при неверных данных', () => {
  cy.visit('http://localhost:8080/login');
  cy.get('input[name="login"]').type('wrong_user');
  cy.get('input[name="password"]').type('wrong_pass');
  cy.contains('Войти').click();
  
  
  cy.contains('Неверный логин или пароль').should('be.visible');
  });
  });
  ```

- Корректное отображения интерфейса после входа

  ```javascript
    describe('Main Page UI Rendering', () => {
    it('Проверка отображения ключевых элементов интерфейса', () => {
    cy.visit('http://localhost:8080/login');
    cy.get('input[name="login"]').type('test_user');
    cy.get('input[name="password"]').type('test_password');
    cy.contains('Войти').click();
    
    
    cy.url().should('include', '/main');
    cy.contains('Создать').should('be.visible');
    cy.contains('Фильтры').should('be.visible');
    cy.get('.table').should('exist');
    });
    });
  ```
